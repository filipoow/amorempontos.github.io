Amor. Em. Pontos.

Imagine a alegria de uma mãe ao receber um kit completo para o seu bebê! O projeto consiste em montar kits especiais contendo uma touca super fofinha para aquecer a cabecinha do bebê, uma manta macia e confortável com o tamanho perfeito para embalar o pequeno, um casaquinho quentinho e charmoso para protegê-lo do frio e um sapatinho delicado para manter os pezinhos sempre aquecidos.
